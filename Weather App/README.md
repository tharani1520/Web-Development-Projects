A simple weather application that fetches and displays real-time weather information using the OpenWeatherMap API.
## Features
- Search weather by city
- Displays:
  - City Name
  - Weather Condition
  - Temperature
- Handles invalid city names

## Technologies Used
- Interactive HTML
- CSS
- JS DOM and events, JS async
- Fetch API
- OpenWeatherMap API

To setup - Clone the repository, generate an API key from OpenWeatherMap API and replace:
  const apiKey = "Your_API_Key' with your API key.
